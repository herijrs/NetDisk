## 基于SpringMVC的网络硬盘
### 项目简介
利用Springmvc+Jpa实现主要功能，前端部分采用ajax提交请求刷新界面，还采用了bootstrap前端框架搭建主要界面和fontawesome图标字体库

### 主要功能
 - 用户注册、登录和基本信息维护；
 - 文件的上传和下载；
 - 文件搜索；
 - 上传文件类型限制；
 - 常见类型文件，例如txt文件、图片文件等的网络预览；
 - 文件分享，类似于百度云盘的文件提取；
 - 用户空间限制
